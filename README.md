# SpringLineAuth
## 概要
### 内容 
LINE認証をSpring Bootで実装するリポジトリ。

### 目的
- Spring Securityの復習
- 認証系の実装
- LINE developerを通して認証できるようにする
  - LINE認証ページへリダイレクト
  - コールバック
  - 認証＆結果によって画面遷移
